Aide Plugin PicasaEmbed
  • Généralité

    • Ce Plugin permet d'afficher vos albums Picasa dans le Picasa Embed. Gérérer vos Picasa Embed. Tous vos albums sont directement utilisables, il ne vous reste plus qu´à les personnaliser et les insérer.

  • Paramétrage

    • La liste des albums est automatiquement mise à jour en synchronisant les albums mémorisés en base. Il est nécessaire de vous identifier (votre compte Google) afin que le plugin sauvegarde votre clef d´authentification en base. Ainsi ce plugin ne mémorise pas votre mot de passe.
    La liste de vos albums étant automatiquements mise à jour il vous est possible d'insérer n'importe quel album dans une page ou post.
    Liste des paramètres

    • Largeur: largeur du Embed
    • Hauteur: hauteur du Embed

  • Utilisation

    • Copier le ' Shortcode ' dans la liste de léalbum souhaité et copier dans la page ou le post cible.

  • Support

    • Pour suggérer une évolution, amélioration ou signaler une anomalie utilisez le formulaire contact en précisant 'WP PicasaEmbed' dans l'objet.
